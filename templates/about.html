About Us

In this project we need to create an application that help the user to establish connect between the local machine and the remote machine and to transfer the files( doctuments , images, pdf etc..,) from master machine to to the worker machine using various tools.

this project we will be using ansible playbooks to connect with the node machines and transfer files across the machines. In this project we are using Angular for the front end and java as a backend. The node and master machines are created by AWS . And establish connection via SSH and controller them using ansible.
